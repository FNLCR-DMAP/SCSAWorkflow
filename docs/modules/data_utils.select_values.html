spac.data_utils.select_values(data, annotation, values=None)[source]

Selects rows from input dataframe matching specified values in a column.

Parameters:

  • data (pandas.DataFrame) – The input dataframe.

  • annotation (str) – The column name to be used for selection.

  • values (list, optional) – List of values for annotation to include. If None, return all values.

Returns:

Dataframe containing only the selected rows.

Return type:

pandas.DataFrame

Raises:

ValueError – If annotation does not exist or one or more values passed do not exist in the specified column.

Examples

>>> df = pd.DataFrame({
...     'column1': ['A', 'B', 'A', 'B', 'A'],
...     'column2': [1, 2, 3, 4, 5]
... })
>>> select_values(df, 'column1', ['A'])
  column1  column2
0       A        1
2       A        3
4       A        5
